Over deze site

Deze site is gebouwd met Pelican.

Pelican is a static site generator, written in Python, that requires no database or server-side logic.

Aangezien ik Python aan het leren ben, onder andere omdat ik geïnteresseerd ben in datavisualisatie en datascience, was dit een goede aanleioding om wat met Python te stoeien. De eerlijkheid gebiedt me te zeggen dat er (nog) niet veel kennis van Python vereist is, de site generator werkt 'out of the box' al direct goed, het enige wat belangrijk is zijn de settings, maar daar hoef je geen Python voor te kennen om dat te snappen.

Het design is gebaseerd op Nest van Molivier.

De gears-settings icon is gemaakt door Freepik van www.flaticon.com en valt onder een CC 3.0 BY license.

De random quote generator is gemaakt met Javascript en is te vinden op Codepen.

Achtergrondplaten komen van Unsplash en zijn kleiner gemaakt met behulp van Bulk resize photos.

De site wordt gehost op Github.

Teksten worden geschreven in Markdown.

Mocht je zelf een statische site willen maken, dan kan ik je ook Jekyll aanraden, hetzelfde idee maar dan geschreven in Ruby.

Jekyll is a blog-aware, static site generator in Ruby

Ik vond het een nadeel dat ik niet goed zag wat de Ruby gems nou precies deden, en daarom heb ik uiteindelijk gekozen voor Pelican. Voordeel van Jekyll is dat de statische pagina's automatisch worden gegenereerd als je de site op Github host.

Naast artikelen en pagina's, heeft Pelican (net als Jekyll overigens) de mogelijkheid van categorieën en tags, zoals je kunt zien bij de verschillende artikelen.
